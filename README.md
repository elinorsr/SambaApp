**A Native Android application for learning and managing Samba dance lessons.**
This academic project demonstrates modern Android development practices using MVVM architecture and cloud integration.

##  Tech Stack
* **Language:** Java
* **Architecture:** MVVM (Model-View-ViewModel)
* **Database:** Firebase Realtime Database
* **UI:** XML Layouts, RecyclerView, Fragments (Tabbed Activity)
* **IDE:** Android Studio

##  Key Features
* **Lesson Management:** Admin capabilities to add, edit, and delete dance lessons.
* **Progress Tracking:** Users can mark lessons as "Favorites" or "Completed".
* **Level System:** Content organized by difficulty (Beginner, Advanced, Expert).
* **Media Integration:** Video player for watching dance steps.

##  Future Improvements
* Refactor Adapter logic for better performance.
